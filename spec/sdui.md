# SDUI Components

## Custom Components

No custom components are defined yet. The initial frontend ships with the base SDUI component registry. Project-specific components will be added as the middleend contract introduces them.

## Screens

Screens are defined by the middleend as SDUI trees. The frontend does not enumerate screens; it renders whatever tree it receives for a given route. Route handling mirrors the paths served by the middleend.

# vk-investment-frontend-v2

Next.js frontend — renders SDUI component trees from the middleend.

## Setup

```bash
cp .env.example .env
corepack enable
pnpm install
```

## Commands

| Command      | Description             |
| ------------ | ----------------------- |
| `make run`   | Start dev server        |
| `make build` | Build for production    |
| `make test`  | Run tests (JSON output) |
| `make lint`  | Run linting checks      |
| `make info`  | Print project metadata  |
| `make clean` | Remove build artifacts  |
